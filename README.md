# True/False Quiz Game  
### 100 Days of Code – Day 17 ✓

A simple yet fully Object-Oriented Programming (OOP) based True/False quiz game.  
It fetches fresh questions every time from the Open Trivia Database API and tracks the user's score.

## Features
- Complete OOP architecture (3 classes: Question model, QuizBrain, and main)
- Real-time question fetching from an API
- User answer validation and score calculation
- Clean, commented, and easy-to-read code

## Project Structure
quiz-game/
├── main.py              # Main program entry point
